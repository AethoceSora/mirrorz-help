> Alpine Linux is a security-oriented, lightweight Linux distribution based on musl libc and busybox.

Alpine Linux 是一个面向安全，轻量级的基于 musl libc 与 busybox 项目的 Linux 发行版。

在终端输入以下命令以替换镜像源：

<CodeBlock>

```bash
sed -i 's#https\?://dl-cdn.alpinelinux.org#{{http_protocol}}{{mirror}}#g' /etc/apk/repositories
```

</CodeBlock>

## postmarketOS

本节供 [postmarketOS 帮助](/postmarketOS/)使用。

<CodeBlock>

```bash
pmbootstrap --mirror-pmOS=http://mirror/postmarketOS/ --mirror-alpine={{http_protocol}}{{mirror}}/ init
pmbootstrap --mirror-pmOS=http://mirror/postmarketOS/ --mirror-alpine={{http_protocol}}{{mirror}}/ install
```

</CodeBlock>

将 `/etc/apk/repositories` 替换为以下的内容

<CodeBlock>

```
http://mirror/postmarketOS/master
{{http_protocol}}{{mirror}}/edge/main
{{http_protocol}}{{mirror}}/edge/community
```

</CodeBlock>
